# 🐾 NutriPet - Site sobre Alimentação Animal

Um site desenvolvido em **HTML5** e **CSS3** com foco em apresentar informações sobre alimentação animal de forma moderna, organizada e responsiva.

## 📸 Prévia

O projeto possui:

- Página inicial com banner.
- Menu de navegação.
- Seção sobre alimentação animal.
- Cards com diferentes espécies.
- Benefícios da nutrição.
- Rodapé com informações de contato.
- Layout responsivo.
